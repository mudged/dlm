# User guide

How to install and operate Domestic Light & Magic as an end user.

  Windows), optional Python, and camera capture.
- [`running-as-a-service.md`](running-as-a-service.md) — run on boot with systemd, and update to a new
  release.
- [`using-the-app.md`](using-the-app.md) — models, scenes, routines, devices, options, and sample
  data.
- [`build-model-from-video.md`](build-model-from-video.md) — estimate light positions from video
  recordings.
