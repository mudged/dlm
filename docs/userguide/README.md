# User guide

Welcome! This guide shows you how to set up and use **Domestic Light & Magic** (or **dlm** for
short). You don't need to be a programmer to follow along — just take it one page at a time.

**What is dlm?** It's a small app that lets you play with strings of smart lights (like the LED
fairy lights or Christmas lights that can change colour). You tell the app where each light sits in
3D space, and then you can see them on your screen, make cool light "scenes", and run animations.

## Where to start

Read these in order the first time:

1. **[Getting started](getting-started.md)** — download the app and open it in your web browser.
2. **[Using the app](using-the-app.md)** — a tour of everything you can do once it's running.
3. **[Build a model from video](build-model-from-video.md)** — let the app figure out where your
   lights are by filming them, instead of typing in numbers.
4. **[Run it automatically](running-as-a-service.md)** — make the app start on its own when a
   Raspberry Pi turns on (a bit more advanced — skip it until you need it).
