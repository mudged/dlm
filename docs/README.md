# Documentation layout

This folder holds **two kinds** of Markdown:

## [`user/`](user/)

Introductory and **how-to** material for people using or lightly customizing the application (installation, troubleshooting, optional deeper notes). Start with the root **[README](../README.md)**.

## [`agentic-development/`](agentic-development/)

Specifications and internal engineering docs used by the **requirements → architect → implementor → verifier** workflow: requirements, acceptance criteria, architecture, traceability matrix, and templates under [`agentic-development/templates/`](agentic-development/templates/).
